Rockman X: The Wily Cut is a Modified project of Mega Man X Online: Deathmatch made by Gamemaker 19

The main focus of this project is to expand the game to new boundaries while also improving it's combat to a hybrid of Mega Man X gameplay and a Fighting game gameplay.

Having anti-virus issues?

Some people have reported that the main Windows download is flagged as malware by Windows Defender or other anti-virus programs. However, there is a large community that plays the game and there is no evidence of any malware being present. Either allow the exe in your anti-virus settings, or try downloading the other Windows versions below (windows 64 bit or non-self contained versions) which are less likely to be flagged as a false positive.

If you have a Mac or Linux installed, download the Mac or Linux versions as they are virtually guaranteed not to have any virus flagging issues since Mac and Linux lack viruses and generally don't use or need an anti-virus.

Alternatively, you can just build the game from source yourself which may eliminate the warnings. Source code can be found here: https://github.com/gamemaker19/MMX-Online-Deathmatch.
